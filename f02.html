王小明的學習目標

以下是本學期資料庫程式設計課程的學習目標，希望透過每週練習，逐步掌握網頁與資料庫的核心技術。

  • 學會 HTML 基本結構與語意化標籤
  • 熟悉 CSS 排版與美化技巧
  • 使用 PHP 撰寫動態網頁
  • 學會 MySQL 資料庫的 CRUD 操作
  • 完成留言板與商品管理系統

學習環境介紹

本課程使用 Laragon 作為本機開發環境，整合 Apache、MySQL 與 PHP，方便同學在自己的電腦上練習網頁開發。

圖：PHP 程式語言標誌

開發工具：VS Code 編輯器 + Laragon 伺服器 + HeidiSQL 資料庫管理工具。
